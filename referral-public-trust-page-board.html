Referral public trust-page placement

Place one referral-safe trust asset on one public trust page without turning it into a broader story than anyone approved.

A referral public trust page works best when it stays compact. The right move is usually one credibility block, one headline, one safe context line, and one clear page destination. That keeps the trust public enough to help conversion while keeping the hidden referral path and hidden details out of view.

Referral Trust Page Rule

A referral public trust page should feel like one precise credibility block, not a public retelling of the full relationship path.

The page block should answer five things fast: what changed, which context line is safe, what stays hidden, where on the trust page it belongs, and when the story needs a bigger format later.

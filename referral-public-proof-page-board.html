Referral public proof-page placement

Place one referral-safe proof asset on one public proof page without turning it into a broader story than anyone approved.

A referral public proof page works best when it stays compact. The right move is usually one proof block, one headline, one safe context line, and one clear page destination. That keeps the proof public enough to help trust while keeping the hidden referral path and hidden details out of view.

Referral Proof Page Rule

A referral public proof page should feel like one precise proof block, not a public retelling of the full relationship path.

The page block should answer five things fast: what the result was, which context line is safe, what stays hidden, where on the proof page it belongs, and when the story needs a bigger format later.

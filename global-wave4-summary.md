# Global Wave 4 Summary

第四波专门打：

- AI creator tools
- newsletter tools
- audience growth tools
- media / publishing workflow tools

这类目标适合快速成交的原因：

- 很依赖 landing page 和 next-step clarity
- 很看重高意向流量的 capture
- 很容易理解 offer tightening、follow-up、handoff 这类工作

## Wave 4 Targets

### 1. Sitekick

- URL: https://www.sitekick.ai/
- 审站结论: no visible web form for lead capture; no clear scheduling or booking step
- 切入角度: 强化 creator / product traffic 的 capture 和 handoff

### 2. Newsletter Compass

- URL: https://www.newslettercompass.com/
- 审站结论: no visible web form for lead capture; no obvious phone CTA above the fold
- 切入角度: 从 newsletter 访客的 capture 和 next-step 承接切入

### 3. Inkredo

- URL: https://inkredo.com/
- 审站结论: no visible web form for lead capture; no obvious phone CTA above the fold
- 切入角度: 强化 creator workflow 工具页的 warm traffic capture

### 4. Zeen

- URL: https://www.yourzeen.com/
- 审站结论: no obvious phone CTA above the fold; no live chat or instant response layer
- 切入角度: 增强高兴趣访问者的即时承接

### 5. Typefully

- URL: https://typefully.com/
- 审站结论: no visible web form for lead capture; no obvious phone CTA above the fold
- 切入角度: 强化 creator / publishing 流量到下一步的 capture

### 6. Creatomate

- URL: https://creatomate.com/
- 审站结论: no visible web form for lead capture; no obvious phone CTA above the fold
- 切入角度: 从内容自动化流量的 high-intent handoff 切入

## 建议优先级

优先先打这 3 个：

1. Sitekick
2. Newsletter Compass
3. Creatomate

原因：

- 与 landing page / conversion / audience growth 的关联更直接
- 你的 sprint 描述更容易被这些团队快速理解
- 比超大平台更容易接受短平快外联

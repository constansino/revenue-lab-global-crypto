# Global Wave 5 Summary

第五波专门打：

- agency
- studio
- service operators
- 已经在卖服务的团队

这类目标适合短单的原因：

- 已经理解服务价值
- 更容易接受固定范围 sprint
- 页面上的承接和跟进问题通常和成交直接相关

## Wave 5 Targets

### 1. Supercharged Studio

- URL: https://www.supercharged.studio/website-design
- 审站结论: no visible web form for lead capture; no clear scheduling or booking step
- 切入角度: web design 服务页的 high-intent handoff

### 2. Arise Web Studio

- URL: https://www.arisewebstudio.com/
- 审站结论: no visible web form for lead capture; no clear scheduling or booking step
- 切入角度: agency 流量到下一步动作的承接

### 3. Digital Growth Studio

- URL: https://www.digitalgrowthstudio.com/
- 审站结论: no clear scheduling or booking step; no live chat or instant response layer
- 切入角度: warm leads 的 capture 和 response layer

### 4. South Digital

- URL: https://south.digital/services/web-design
- 审站结论: no visible web form for lead capture; no clear scheduling or booking step
- 切入角度: 服务页从兴趣到咨询动作的承接

### 5. Ish Studio

- URL: https://www.ish.studio/services/growth-support-package
- 审站结论: no visible web form for lead capture; no live chat or instant response layer
- 切入角度: package page 的 warm traffic capture

### 6. Upgrow

- URL: https://www.upgrow.io/pricing
- 审站结论: no visible web form for lead capture; no live chat or instant response layer
- 切入角度: pricing page 后的 handoff 和 response speed

## 建议优先级

优先先打这 3 个：

1. Supercharged Studio
2. Digital Growth Studio
3. Upgrow

原因：

- 服务价值更容易直接关联到你的 sprint
- 页面问题更贴近“更强 handoff / quote / follow-up”这种短单服务
- 比纯工具团队更可能快速理解并购买服务型修正

Reply Rule

Classify first. Reply second.

Do not draft from scratch. Identify the reply type, use the matching response, then move to teardown, offer, or payment only when the conversation has earned that step.

Reply Cases

Match the reply type, then send the next line.

Positive Interest

Makes sense.

The short version is that I see one higher-friction step between interest and the next action.

I can send a tight 3-point teardown first, and if it feels useful, package the fix as a short sprint.

Ask For Details

The practical version is:

1. identify the highest-friction step
2. tighten the page or next-step structure
3. tighten the follow-up or handoff path

I usually keep it narrow so it is easy to buy and easy to ship.

Ask For Price

Usually this fits one of three buckets:

`$299` for a short teardown
`$799` for the core sprint
`$1,500` for a fuller setup

If useful, I can point to which bucket your current setup fits.

Advance Path

When interest is real, move through the same sequence every time.

1. Send Mini Teardown

The short version:
your offer already makes sense
the leak is mostly in the handoff from interest to the next action
I would tighten that first before changing broader messaging

2. Send Scoped Sprint

Keep it narrow:
identify the highest-friction step
tighten the page or pricing handoff
tighten the next-step or follow-up structure

3. Ask For Deposit

If that scope works, the next step is the deposit.

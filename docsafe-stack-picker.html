Stack Picker

Match the stack to the workflow pressure you already have.

Use advanced if several owners, sequence control, or stronger audit trail matters.

Use high if teams constantly merge, split, redact, OCR, or repackage files.

Choose yes if requests still arrive via loose email threads, chats, or attachments.

Heavy means multiple apps, multi-step state changes, or follow-up rules.

Use multi when the workflow crosses departments or approval owners.

This affects whether the signing and automation choice should bias toward control or speed.

Client-referral public proof page

Place one referral win on a public proof page without turning it into a broader story than anyone approved.

A referral-backed proof page works best when it stays compact. The strongest version is one proof block, one headline, one safe context line, and one clear page destination. That keeps the proof public enough to help the next buyer while keeping the original referrer, buyer path, and hidden details out of sight.

Referral Proof Rule

A referral public proof page should feel like one precise proof block, not a public retelling of the whole deal.

The page block should answer five things fast: what shifted, which context line is safe, what stays hidden, where on the proof page it belongs, and when the asset should widen later only by decision.

Good Proof Blocks

Most referral public proof pages should start with one of these smaller shapes.

Anonymous Result Strip

One line on buyer type, bottleneck, and visible shift, without exposing who the buyer or referrer actually was.

Brandless Before-After Card

A compact before-after summary that travels well on a proof page without screenshots or traceable brand detail.

Quote-Led Proof Block

One approved line with a short context sentence when a voice cue adds trust without needing a larger public story.

Trust-Focused Proof Note

One public-safe line that supports credibility without turning into a broader narrative about the whole deal.

Do-Not-Widen Block

A public proof block that stays useful because it does not drift into every page, pack, and site claim at once.

Short Lines

Use short lines that keep the referral public proof page compact and accurate.

Placement Line

I want this to live as one public proof block, not as a broader public story, so the placement should stay narrow.

Headline Line

The headline should name the bottleneck and the shift so the page is clear fast without retelling the full private path.

Boundary Line

Names, screenshots, deal mechanics, and the original referral path stay outside this public block unless cleared separately.

Forward Line

This page should help the next referral move faster, not become a detached public asset with no clear sales route.

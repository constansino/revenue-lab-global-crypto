# China Profit Map

面向中国大陆可访问环境，优先选择能直接注册、直接收款、直接开始卖的路径。

## 1. 现金流最快：卖 AI 工作流代搭服务

适合你现在立刻开卖。

### 平台

- 程序员客栈: https://www.proginn.com/
- 猪八戒网: https://www.zbj.com/

### 为什么

- 程序员客栈首页直接提供 `开发者入驻`、`发布需求`、`云端工作` 入口。
- 猪八戒首页直接提供 `免费开店赚钱`、`任务大厅`、`AI人工智能` 类目。

### 卖什么

- 飞书多维表格业务流搭建
- 扣子工作流/Agent 搭建
- 表单收集 + 自动提醒 + 客户跟进 + 报价流转

### 推荐报价

- `699 元`：单个流程梳理 + 页面文案 + 1 个自动化
- `1999 元`：完整工作流落地 + 培训文档
- `4999 元`：按行业定制 + 7 天陪跑

## 2. 复利更强：卖模板和会员内容

适合把重复交付抽成模板包、案例包、操作手册。

### 平台

- 知识星球: https://www.zsxq.com/
- 爱发电: https://afdian.com/ （2026-03-18 我当前环境连通性失败，作为备选，不作为主路径）

### 为什么

- 知识星球支持付费星球、微信支付、3 日后提现，适合做案例库和陪跑社群。
- 爱发电支持会员制、商品、开发者 API，但我当前环境没有验证通畅，先放备选。

### 卖什么

- 飞书多维表格模板包
- 扣子工作流模板包
- 本地商家获客 SOP
- AI 自动化报价模板

### 推荐售价

- `29-99 元`：单模板
- `199-399 元`：模板包
- `699-999 元/年`：会员制案例库

## 4. 最稳的私域入口：微信内容获客

### 平台

- 微信公众平台: https://mp.weixin.qq.com/
- 微信视频号入口: https://channels.weixin.qq.com/

### 为什么

- 在中国大陆可直接访问，适合沉淀案例、教程、成交截图和私域线索。
- 对本地商家和中小团队来说，微信沟通成本最低，转介绍更自然。

### 发什么

- 行业案例拆解
- 自动化前后对比
- 模板演示短视频
- 成交复盘和报价逻辑

## 建议顺序

1. 先在程序员客栈和猪八戒卖服务，抢第一笔现金流。
2. 把交付过 3 次以上的方案抽成模板，放到爱发电。
3. 把持续答疑、案例拆解和更新日志放到知识星球做复购。
4. 用公众号和视频号持续导流到你的服务页和微信私聊。

# Reply Matrix

把常见回复压成固定动作，减少临场犹豫。

## 1. Positive Interest

典型回复：

- sounds interesting
- sure, send it
- what did you notice

回复：

Makes sense.

The short version is that I see one higher-friction step between interest and the next action.

I can send a tight 3-point teardown first, and if it feels useful, package the fix as a short sprint.

## 2. Ask For Details

典型回复：

- what exactly do you do
- what would you change
- can you be more specific

回复：

The practical version is:

1. identify the highest-friction step
2. tighten the page or next-step structure
3. tighten the follow-up / handoff path

I usually keep it narrow so it is easy to buy and easy to ship.

If useful, I can point to the exact step I would start with on your current page.

## 3. Ask For Price

典型回复：

- how much
- what do you charge
- budget?

回复：

Usually this fits one of three buckets:

- `$299` for a short teardown
- `$799` for the core sprint
- `$1,500` for a fuller setup

The right one depends on whether the issue is mostly handoff, follow-up, or a broader offer flow.

If useful, I can point to which bucket your current setup fits.

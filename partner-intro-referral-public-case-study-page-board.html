Client-referral public case page

Turn one closed referral win into a public case-study page without exposing the hidden referral path behind it.

A referral-backed case page can help future demand, but it should stay narrow. The strongest version is one bottleneck, one fix, one visible shift, and one page destination. That gives a future buyer useful public proof while keeping the original referrer, buyer context, hidden economics, and operating details out of sight.

Referral Case Rule

A referral case-study page should sell the result, not publish the relationship path that created it.

The page should answer five things fast: what changed, why the shift matters, what remains hidden, what proof format is safe, and how the page supports the next referral or repeat conversation.

Case Shapes

Most referral public case pages should start from one of these tighter story formats.

Anonymous Mini Case

Show the buyer pattern, the bottleneck, and the visible shift without exposing names or a traceable referral path.

Brandless Before-After

Show the problem and the result without logos, screenshots, or context that reveals the original buyer too easily.

Quote-Led Referral Story

Use one approved line and one context sentence when a voice cue helps the page without widening it into a bigger testimonial set.

Proof-Backed Mini Page

Use one safe proof block to anchor the story when plain summary alone would feel too soft.

Do-Not-Sprawl Page

Keep the page valuable by not letting it expand into every public asset, outbound forward, and proof slot at once.

Short Lines

Use short lines that keep the public referral case page commercially useful and narrow.

Need Line

We only need one narrow public case page here, not a broader public story that exposes the original referral path.

Spine Line

I want this page to follow one before-after spine so the result is clear without publishing the whole delivery timeline.

Hidden Layer Line

Names, screenshots, deal mechanics, and the original referrer should stay outside the case page unless they are cleared separately.

Forward Line

This page should help the next referral move faster, not become a detached marketing asset with no clear sales route.
